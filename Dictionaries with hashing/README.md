# Hash Table Duel: Chaining vs. Open Addressing

Projekt edukacyjny w Pythonie prezentujący dwie klasyczne implementacje tablicy haszującej:
- Separate Chaining (łańcuchowanie)
- Open Addressing z linear probing

Celem projektu jest pokazanie, jak działają kolizje, usuwanie elementów oraz rehashing bez użycia wbudowanego `dict`.

## Zawartość

- Implementacja tablicy z łańcuchowaniem
- Implementacja tablicy z adresowaniem otwartym
- Operacje: insert, find, remove
- Dynamiczna zmiana rozmiaru tablicy
- Wypisywanie stanu tablicy w konsoli

## Technologie

- Python 3.x
- Standardowa biblioteka Pythona
